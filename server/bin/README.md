# yt-dlp binary

Place the appropriate `yt-dlp` binary here for your platform:

- **Linux/Mac**: `yt-dlp` (make executable: `chmod +x yt-dlp`)
- **Windows**: `yt-dlp.exe`

Download from: https://github.com/yt-dlp/yt-dlp/releases

The binary is excluded from version control via `.gitignore`.
Keep it updated regularly to maintain platform compatibility.
